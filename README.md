# GCounter
A simple chrome extension to find how many times you Google a day. 

### How to install this Extension?

This extension is currently not available in chrome web store. You can install the extension by following these simple steps.

1. Download the repository.
2. Navigate to `chrome://extensions/` in your chrome browser.
3. Enable `Developer Mode`, it is a checkbox at the top.
4. Click `Load Unpacked Extension`, browse to the downloaded repository and click Ok.
